# Futura

## Description

Futura is a secure, decentralized digital vault built on the Internet Computer Protocol (ICP), designed to preserve and share your most important life stories, photos, videos, and documents across generations. By leveraging ICPâ€™s decentralized infrastructure, Futura ensures that your personal memories are stored securely and remain accessible for the future. It offers a user-friendly platform where users can create, safeguard, and pass down their legacies with confidence.

## Technology Stack

- **Backend:** Rust, Internet Computer Protocol (ICP)
- **Frontend:** React, TypeScript, Vite, Shadcn UI
